# AI 内容生产与分发平台文档

本目录用于维护 AI 内容生产与分发平台的需求、竞品分析和后续阶段计划。

## 文档列表

- [一期需求计划书](phase-1-requirement-plan.md)
- [开发路线](development-roadmap.md)
- [竞品分析](competitor-analysis.md)
- [真实大模型接入说明](real-model-integration.md)

## 文档职责

`phase-1-requirement-plan.md` 是一期详细需求源，维护：

- 一期目标
- 功能范围
- 页面规划
- 数据对象
- 数据表初稿
- 接口初稿
- 里程碑
- 验收标准
- 非一期范围

`development-roadmap.md` 用于维护：

- 当前第一步
- 后续开发顺序
- 每个阶段的前后端工作
- 每个阶段的可见产出
- 每个阶段的验收标准
- 下一次开发建议

`competitor-analysis.md` 用于维护：

- 竞品分类
- 重点竞品分析
- 能力对比
- 市场机会
- 差异化定位
- 一期、二期、三期产品策略

`real-model-integration.md` 用于维护：

- 真实大模型接入配置
- OpenAI 兼容接口约定
- 本地启动方式
- 当前模型调用边界

## 与顶层路线图的关系

AI 产品套件的顶层规划见 [AI 双系统产品规划](../ai-product-suite/dual-system-roadmap.md)。

顶层路线图只描述 Agent 工作台、AI 内容平台和共享 AI 能力层的关系。AI 内容平台一期落地细节以本目录的 `phase-1-requirement-plan.md` 为准。

具体开发顺序以 [开发路线](development-roadmap.md) 为准。
